# Smart Glass IoT Project

## Overview
An IoT-based Smart Glass system developed as a group project to demonstrate smart wearable technology using embedded systems and sensor integration.

## Features
- Obstacle detection
- Real-time monitoring
- Sensor integration
- Smart wearable prototype

## Technologies Used
- IoT
- Arduino / ESP32
- Embedded Systems
- HTML/CSS

## Team
Group project developed by a team of four members.
